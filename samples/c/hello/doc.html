C Hello Sample

This is the simplist C sample. It ensures everything is installed correctly.

Windows

  1. Compile and link hello.pro into hello.xpl.
    1. acmp hello
    2. alnk hello hello
  2. Under Windows, open the Microsoft Visual C Project hellop and build hello.exe.
  3. Run hello(.exe)

Unix/Mac

  1. Under Unix/Mac, run the appropriate makefile to build hello
    1. make -f make_X.txt
  2. Run: ./hello
