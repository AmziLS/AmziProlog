C++ Pets Sample

This sample demonstrates calling a simple expert system from C++.

Windows

  1. Compile and link pets.pro into pets.xpl.
    1. acmp pets
    2. alnk pets pets
  2. Under Windows, open the Microsoft Visual C++ Project pets and build pets.exe.
  3. Run pets(.exe)

Unix/Mac

  1. Run the appropriate makefile build pets
  2. run: ./pets
